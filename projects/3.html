Create a subpage

1. Create a new file

  1. From the 'Code' page on GitHub, click 'create new file':
  2. Name the page. And don't forget the .html at the end!

2. Create a new html file (with whatever name you want) and commit it!

3. New page will be at yoursite.xyz/pagename

Link to the subpage

1. Make a link using an 'a href' tag.

It should look like this!

<a href="http://yoursite.xyz/yourpagename">any name you want</a>

2. That's it!
